
# LUBNA-DEV  Recipe Recommendation System

DishDiscovery is an interactive web application designed to reduce food waste and solve the daily "what should I cook?" dilemma. Users can input the ingredients they already have in their fridge, apply dietary filters, and instantly receive personalized recipe recommendations with dynamic ingredient scaling.

## 🚀 Prompt Used for Generation

This entire application was generated using a text-to-code AI creator. Below is the exact prompt utilized to build the system:

> "Build a modern, responsive web application called 'DishDiscovery' – an AI-powered recipe recommendation system. The UI should look clean and minimalist, using a soft warm color palette (cream backgrounds, terracotta accents, and dark charcoal text).
>
> **Key Features and Pages to Include:**
> 1. **Landing Page / Dashboard:** A prominent search bar for ingredients, toggle filters for dietary restrictions (Vegan, Keto, etc.), cooking time, and skill level.
> 2. **Recipe Results Page:** A grid of cards showing placeholder images, matching scores, titles, and cooking times.
> 3. **Detailed Recipe View:** A modal/page detailing ingredients owned vs. missing, step-by-step interactive checkboxes, and a dynamic servings calculator.
> 4. **Surprise Me:** A shuffle button that recommends a random, highly-rated quick recipe."

---

## ✨ Features

- **Ingredient-Based Search:** Input available ingredients to find matching recipes.
- **Dynamic Matching Score:** Shows a percentage breakdown of how well a recipe matches your current inventory.
- **Smart Dietary Filters:** Filter by Vegan, Vegetarian, Gluten-Free, Keto, and Dairy-Free.
- **Interactive Cooking Mode:** Check off steps as you cook to easily track your progress.
- **Dynamic Servings Calculator:** Automatically scales ingredient quantities up or down based on desired servings.
- **Local Storage Persistence:** Remembers user dietary preferences even after refreshing the page.

---

## 🛠️ Technology Stack (Generated)

- **Frontend:** HTML5, Tailwind CSS / Custom CSS, JavaScript (ES6+)
- **State Management:** LocalStorage API for persisting user preferences.
- **UI Paradigm:** Responsive Grid & Component-Driven Architecture.
